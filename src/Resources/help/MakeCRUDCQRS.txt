The <info>%command.name%</info> command generates all classes related to PrestaShop CRUD Forms.
Get more informations about CQRS in the developer documentation: https://devdocs.prestashop.com/1.7/development/architecture/domain/cqrs/
Check also its usage in CRUD forms: https://devdocs.prestashop.com/1.7/development/architecture/migration-guide/forms/cqrs-usage-in-forms/

<info>php %command.full_name% EntityClassName</info>

If the argument is missing, the command will ask for the entity class name interactively.
